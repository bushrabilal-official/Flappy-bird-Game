# 🐦 Flappy Bird Game

A simple and fun browser-based Flappy Bird game built using HTML, CSS, and JavaScript. Control the bird, avoid obstacles, and challenge yourself to achieve the highest score while enjoying smooth gameplay and responsive controls.

---

## 🎮 Game Features

* 🐤 Classic Flappy Bird gameplay
* 🎯 Endless obstacle generation
* 🏆 Real-time score tracking
* ⚡ Smooth animations and game mechanics
* 🎮 Easy keyboard controls
* 🔄 Restart game after Game Over
* 📱 Responsive design for desktop and mobile
* 🎨 Clean and lightweight interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

### Clone the repository

bash
git clone https://github.com/YOUR_USERNAME/flappy-bird.git


### Run the Game

1. Clone or download the repository.
2. Open the project folder.
3. Double-click *index.html* or open it in your web browser.
4. Press *Space* (or click/tap if supported) to start playing.

---

## 🎮 Controls

* *Space Bar* — Make the bird jump
* *Mouse Click / Tap* (if supported) — Jump
* Avoid hitting pipes or the ground.
* Try to beat your highest score!
